# University Website Project

This is a PHP web application developed for a university website. It uses Composer for managing dependencies (like PHPMailer).

## Setup
1. Clone the repository.
2. Run `composer install` to install dependencies.
3. Serve the `Public` directory using your preferred web server (e.g., Apache, Nginx) or XAMPP.
4. Access `index.php` in your browser.

## Structure
- `assets/`: Contains JS, CSS, images, and PHP helper scripts.
- `config/`: Configuration files for database and other settings.
- `pages/`: Additional PHP/HTML pages for the university application.
- `layout/`: Shared layout components (header, footer, etc.).
